# SAGE - Seating Arrangement Generator Engine

Générateur automatique de plans de classe optimisés.

## Installation

```bash
poetry install
poetry run python main.py
```

## Documentation

La documentation complète est disponible dans le dossier `docs/` :
- **Guide de démarrage** : `docs/getting-started.rst`
- **API** : `docs/api.rst`
- **Introduction** : `docs/intro.rst`

Générer la documentation HTML :
```bash
cd docs
make html
```
